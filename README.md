# 育成バランス配置シミュレーター

施設の要求技芸と、知己の固定技芸をもとに  
店長・副店長・補欠を自動配置し、育成まで行うWEBツールです。

## 特徴
- 技芸一致数を最優先
- 合計レベルが低い知己を優先（育成重視）
- 店長100% / 副店長15% 経験値獲得
- 店長・副店長は兼任不可
- サーバー不要（HTML1枚）

## 使い方
1. index.html をブラウザで開く
2. 保有している知己にチェック
3. 合計Lvを入力
4. 「配置・育成を実行」を押す

## GitHub Pages
このリポジトリを GitHub Pages に設定すれば  
そのままWEBアプリとして公開できます。
